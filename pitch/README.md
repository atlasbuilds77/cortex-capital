## 📊 CONVERSION TOOLS

### Convert Markdown to Slides

**Recommended tools:**

1. **Pitch.com** (Best for startups)
   - Import markdown or recreate visually
   - Collaborative editing
   - Beautiful templates
   - Free tier available

2. **Google Slides** (Best for sharing)
   - Familiar interface
   - Easy sharing/commenting
   - Export to PDF/PowerPoint

3. **PowerPoint** (Best for VCs)
   - Standard format
   - Works offline
   - Professional templates

4. **Keynote** (Best for design)
   - Apple ecosystem
   - Beautiful animations
   - Export to PDF/PowerPoint

**Process:**
1. Copy slide content from PITCH-DECK.md
2. One slide = one `## SLIDE X` section
3. Add visuals (screenshots, charts, logos)
4. Keep text minimal (presenter notes = markdown detail)

---

### Create PDF One-Pager

**Tools:**
- **Markdown to PDF:** Pandoc, Marked 2, Typora
- **Design tools:** Canva, Figma, Adobe InDesign

**Tips:**
- Use company colors/branding
- Add logo at top
- Include contact info at bottom
- Keep to 1-2 printed pages
- Export as high-res PDF
