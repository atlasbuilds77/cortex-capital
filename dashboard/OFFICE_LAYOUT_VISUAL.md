## Implementation Notes

### Grid System
- **Tile size:** 64x32 pixels (isometric diamond)
- **Total grid:** ~20x15 tiles
- **Walkable tiles:** Mark in grid data
- **Obstacle tiles:** Desks, trees, servers

### Z-Index / Layering
1. **Floor tiles** (z=0)
2. **Desk bases** (z=1)
3. **Environmental decorations** (z=2)
4. **Characters** (z=3, varies by Y position)
5. **Particle effects** (z=4)
6. **UI overlays** (z=5)

### Character Spawn Points
- Each character starts at their desk
- Idle animation plays
- On event trigger ‚Üí walk to destination ‚Üí animate ‚Üí return

### Camera View
- **Default:** Centered on Atlas roundtable
- **Zoom:** Show all 13 desks
- **Static:** No camera movement (as requested)
- **Click:** Zoom to clicked desk (optional)
